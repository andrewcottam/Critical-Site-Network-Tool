The SiteSearchBox is an MXML component that is used in the CSN tool to search for a site.

The life cycle of the SiteSearchBox is the following:

  1. The page is loaded and a query is run on initialisation to load a batch of records.
  2. The user enters search text in the box and a query is run to get the matching records.
  3. The list is populated with the matching items.
  4. If a filter is applied, then the filtered results are queried and populated in the list.

protected function searchBox_keyUpHandler(event:KeyboardEvent):void

Handles a key up event on the keyboard. If the CR key is pressed and there is only one item in the list of sites, then this site is selected and the site page for that site is loaded.

If it is not a CR key then the site list is queried for the matching text string.
