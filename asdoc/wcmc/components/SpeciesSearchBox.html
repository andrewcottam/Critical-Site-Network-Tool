The SpeciesSearchBox is an MXML component that is used in the CSN tool to search for a species. The component uses asynchronouse calls to web services to get the data to populate the search box.

The life cycle of the SpeciesSearchBox is the following:

  1. The page is loaded and a query is run on initialisation to load the records. If the user has selected a different language then the species common names will be returned in that language.
  2. The user enters search text in the box and the data in the list box is filtered.
  3. The list is populated with the matching items.
  4. If a filter is applied, then the filtered results are also filtered.

searchBox_keyUpHandler()method 
protected function searchBox_keyUpHandler(event:KeyboardEvent):void

Handles a key up event on the keyboard. If the CR key is pressed and there is only one item in the list of species, then this species is selected and the species page for that species is loaded.

If it is not a CR key then the species list is filtered for the matching text string.
